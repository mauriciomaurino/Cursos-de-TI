com o node.js instalado, abrir o cmd para poder compilar e gerar uma versão do bootstrap com alterações nas variaveis. Primeiro dar o comando 'npm install -g grunt-cli', para instalar o grunt. Depois 'npm install', para instalar o npm. Para alterar as variaveis do Bootstrap, entrar no arquivo baixado do Bootstrap, opção Source code, na pasta less, abrir com editor o arquivo variables. Após alterar o que desejar, salvar, e compilar com o comando 'grunt dist' na pasta booststrap-x.x.x, com isso os arquivos de dentro da pasta do bootstrap serão compilados com as alterações, e quando utilizados no projeto terão efeito
